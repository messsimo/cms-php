## Installation

1. Clone the repository:
    ```
      git clone https://github.com/messsimo/cms_php.git
    ```
2. Go to the project directory:
    ```
      cd cms_php
    ```
3. Install Composer:
    ```
        composer install
    ```
3. Setting up the .env file:
    ```
        cp .env.example .env
    ```
3. Running Migrations:
    ```
        php artisan migrate
    ```
3. Starting the server:
    ```
        php artisan serve
    ```
